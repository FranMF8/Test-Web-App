# Sample Project

This repository contains the source code for a project developed using .NET 6, Angular, SQL Server Express, and Entity Framework.

## Contributors

- Backend: Francisco Matías Fernández
- Frontend: Ana Lucía Beneítez Sánchez and Magali Lucía Rulín

## Requirements

- Visual Studio 2022
- Visual Studio Code
- .NET 6 SDK
- Node.js
- SQL Server Express

## Backend Instructions

1. Open Visual Studio 2022.
2. Clone this repository to your local machine.
3. Open the solution (.sln) file of the project.
4. Make sure the backend project is selected as the startup project.
5. Restore the NuGet packages if necessary.
6. Configure the connection string to the SQL Server Express database in the corresponding configuration file.
7. Run the application by clicking the "Start" button in Visual Studio 2022.

## Frontend Instructions

1. Open Visual Studio Code.
2. Open the frontend project folder.
3. Open a terminal in Visual Studio Code.
4. Make sure you have Node.js installed on your machine.
5. Run the following command in the terminal to install the project dependencies:

npm install

6. Once the dependencies are installed, run the following command in the terminal to start the development server:

ng serve -o
