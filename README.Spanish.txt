# Proyecto Ejemplo

Este repositorio contiene el código fuente de un proyecto desarrollado con .NET 6, Angular, SQL Server Express y Entity Framework.

## Colaboradores

- Backend: Francisco Matías Fernández
- Frontend: Ana Lucía Beneítez Sánchez y Magali Lucía Rulín

## Requisitos

- Visual Studio 2022
- Visual Studio Code
- .NET 6 SDK
- Node.js
- SQL Server Express

## Instrucciones para el Backend

1. Abre Visual Studio 2022.
2. Clona este repositorio en tu máquina local.
3. Abre el archivo de solución (.sln) del proyecto.
4. Asegúrate de que el proyecto de backend esté seleccionado como proyecto de inicio.
5. Restaura los paquetes NuGet si es necesario.
6. Configura la cadena de conexión a la base de datos SQL Server Express en el archivo de configuración correspondiente.
7. Ejecuta la aplicación presionando el botón de "Iniciar" en Visual Studio 2022.

## Instrucciones para el Frontend

1. Abre Visual Studio Code.
2. Abre la carpeta del proyecto de frontend.
3. Abre una terminal en Visual Studio Code.
4. Asegúrate de tener Node.js instalado en tu máquina.
5. Ejecuta el siguiente comando en la terminal para instalar las dependencias del proyecto:

npm install

6. Una vez que se hayan instalado las dependencias, ejecuta el siguiente comando en la terminal para iniciar el servidor de desarrollo:

ng serve -o
